What is it?

A module/mixin system written in the Lua programming language.

Modern

The idea is to add a simple interface for introducing inheritance to native Lua, as well as, including the ability for adding mixins to your modules. The result is a cross between OOP and ECS (Entity-Component-System) methodologies.

Technologies used
